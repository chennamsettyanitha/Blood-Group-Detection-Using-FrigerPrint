This project aims to predict a person’s blood group using fingerprint images by leveraging Deep Learning (CNN) techniques.
Instead of traditional blood testing, the system analyzes fingerprint patterns and classifies them into 8 blood groups:

A+, A-, B+, B-, AB+, AB-, O+, O-

The model is trained on a labeled fingerprint image dataset and achieves ~95% test accuracy, demonstrating strong classification performance.# Blood-Group-Detection-Using-FrigerPrint
